# AI Chatbot

## Overview

This is an advanced Java console-based AI Chatbot developed as part of the CodeAlpha Java Programming Internship.

The chatbot interacts with users using predefined conversational responses and provides multiple utility features.

## Features

* Greeting interaction
* Personalized username memory
* Time response
* Joke generator
* Calculator feature
* Help command
* Continuous conversation loop
* Exit command

## Technologies Used

* Java
* Eclipse IDE

## Concepts Used

* Scanner Class
* String Handling
* Loops
* Conditional Statements
* User Interaction
* Java Libraries
